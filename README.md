# AI算力管理平台 - 设计文档

## 项目概述

半导体数字化与IT装备部AI算力管理平台，用于统一管理红区和黄区两个网络区域的AI算力资源，提供算力申请、令牌分配、运营看板等功能。

**当前版本为嵌入式应用，无需登录即可使用（默认以管理员身份运行）。**

## 技术栈

- **前端框架**: 纯HTML + JavaScript (ES6+)
- **样式**: Tailwind CSS (CDN)
- **图表库**: Chart.js (CDN)
- **UI组件**: 自定义Toast通知系统、Modal模态框
- **特点**: 无需构建工具，可直接在浏览器中运行

## UI风格
- 使用前端设计技能，浅色系

| 功能模块 | 说明 |
|---------|------|

| Modal模态框 | 统一的弹窗组件，用于表单和确认操作 |

| 时间筛选器 | 支持按月份筛选（近1月/近3月/近6月/近1年/自定义） |
| 额度管理Tab切换 | 整合为Tab模式：我的申请/配额审批/用户额度 |
| 运营看板集成视图 | 整合人员/项目/模型维度，支持一键切换 |
| 模型管理页面 | 厂商筛选按钮、模型列表、状态筛选 |
| 数据导出功能 | 支持用户额度、项目、模型等数据CSV导出 |
| 分页功能 | 详细表格支持分页展示 |
| 批量编辑 | 用户额度页面支持批量选择和编辑 |
| 多级部门 | 用户算力明细支持5级部门结构 |
| 模型调用次数 | 模型管理显示调用次数，支持排序 |
| 使用日志筛选 | 支持按时间/令牌/模型/用户筛选 |
| 提示/补全Token | 使用日志显示输入输出Token数 |

- **新建令牌**: 弹出表单，输入名称和过期时间，实时添加到列表
- **吊销令牌**: 确认对话框，吊销后该行变灰显示
- **添加用户**: 弹出表单，输入姓名，部门、额度，实时添加
- **编辑额度**: 弹出表单修改用户Token额度

#### 项目挂靠规则
- 用户首次申请额度时选择项目，挂靠关系从审批通过时确立
- 切换项目时只能选择有剩余额度的项目
- 切换后原项目余额保留（由项目经理分配）
- 挂靠历史记录保存历次变更

## 交互说明

### 导航切换
- 点击左侧导航切换页面
- 额度管理为可折叠菜单，支持展开/收起
- 当前页面高亮显示
- 普通用户自动隐藏管理员菜单

### 运营看板
- 点击"算力使用全景看板"/"项目算力使用明细"切换视图
- 部门/项目/模型/人员维度切换展示不同数据
- 时间范围选择器更新图表数据

### Toast通知
- 操作成功后右上角弹出提示
- 4种类型：成功(绿)/错误(红)/警告(黄)/信息(蓝)
- 3秒后自动消失

### Modal模态框
- 表单操作弹出模态框
- 支持确认/取消操作
- 点击遮罩或关闭按钮可关闭

### 数据导出
- 支持用户额度、项目、模型等数据导出为CSV文件
- 导出按钮位于各列表页面顶部

### 分页
- 详细表格底部显示分页控件
- 支持上一页/下一页/页码跳转
